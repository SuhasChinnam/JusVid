📺 JusVid - Video Sharing Platform
JusVid is a full-stack video sharing web application developed using Spring Boot, Thymeleaf, and MySQL.
It enables users to upload, view, and manage videos securely.

🚀 Features
User Roles:

User: Can upload, view, and delete only their own videos.

Admin: Can view and delete all videos uploaded by any user.

Authentication & Authorization:

Spring Security with role-based access control.

Custom login/logout pages.

Video Management:

Upload videos (stored in /uploads/ directory).

View all public videos.

Manage personal or platform-wide videos based on role.

Database Integration:

MySQL used for storing user, role, and video metadata.

UI:

Built with Thymeleaf templates.

Simple and responsive video gallery layout.
